
# Unemployment Analysis with Python

## Objective
To analyze unemployment trends in India and study the impact of COVID-19 on unemployment rates.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- Trend Analysis
- State-wise Analysis
- Correlation Heatmap

## Key Findings
- Unemployment increased sharply during COVID-19.
- Significant differences were observed among regions.
- Employment and unemployment showed a weak negative correlation.

## Conclusion
This project analyzed unemployment patterns in India and highlighted the economic impact of the COVID-19 pandemic.
